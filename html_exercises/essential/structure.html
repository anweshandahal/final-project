a. What is a markup?
Ans: Markup is all about structuring documents so that it can be manipulated by programs such as web browsers or screen readers.

b. What is hypertext and hypermedia?
Ans: Hypertext is defined as text that contains links to other texts.
Hypermedia is a similar concept and includes graphics, sound and video.

c. What is a doctype and why is it important to include it in every HTML document?
Ans: A doctype declaration specifies the version of HTML being used and in the absence of a doctype browsers will interpret our markup with a different logic, known as the ‘quirck mode’ with possible unexpected results.

d. What language can be used to style web documents?
Ans: Casacading Style Sheets (CSS) is used to style web documents.

e. Which elements are required for a valid XHTML5 document? Ans: Elements are: Doctype, html, head, body, title.

f. What is a self closing HTML element? Give an example? Ans: Self closing tags are the tags that have all the data they need in attribute values and are self-closing without an end tag, for example, the LINK TAG.

g. What is XML and why is it important to web developers? Ans: XML, a general purpose markup language with stricter syntax rules than HTML. It is important to developers because it is essential for well structured documents that are easier to develop and maintain.

h. Describe 4 XML rules?
Ans: The 4 XML rules are
1. Elements must be properly nested.
2. Elements must have a start and end tag.
3. Elements names are case-sensitive.
4.Attributes may not be repeated.

h. What is the difference bwtween a "well-formed" and a valid web document?
Ans: Well formed are the documents adhering to XML and are ideal for manipulation by computer programs such as browsers whereas a document that conforms to a particular doctype and is also "well-formed" is considered a valid document.
